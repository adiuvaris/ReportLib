# ReportLib
## Install

Via Composer

``` bash
$ composer require adiuvaris/reportlib
```

## Usage

``` php
<?php
include_once "src/Report.php";
use Adi\ReportLib as ReportLib;
$rp = new ReportLib\Report();
$rp->output(__DIR__ . "/test.pdf");
```
